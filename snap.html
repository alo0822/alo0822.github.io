Snap! Website

Jan~Dec 2015

UI Design

Snap! is a visual, drag-and-drop programming language. The Snap! Student Development Team began developing a new website that would serve as an online community for Snap! users, and also provided a platform for teachers to create classes online. From January to December 2015, I was the Design Lead for the dev team and was responsible with designing and creating mockups for the website.

OVERVIEW

What: An online community for Snap! users

Goals:

  • Allow users to create, learn, and share Snap! projects
  • Provide a platform for teachers to create classes that used Snap!

Users: High school and college students and teachers.

Tools used: Photoshop, Illustrator, HTML&CSS, Google MDL

My Role: Created designs, mockups, and prototype

HOME PAGE

For the home page, we wanted the design to be simple. The main feature of the home page was to showcase a selection of featured projects that were converted from BYOB3 and Scratch.

Snap! home page

GENERAL/STUDENTS

Users with general or student accounts are able to enroll and participate in classes, create projects, follow users, and more.

Snap! pages for general/student users:
Top: Project view, user projects, user classes
Bottom: Class assignment, all assignments, class participants

INSTRUCTORS

Features of instructor accounts include the ability to create and manage classes, create assignments, and grade submissions.

Snap! pages for instructor users:
Class assignments, class participants, enrollment requests

REFLECTION

Unfortunately, we did not finish the website while I was on the development team as there were some setbacks regarding the back end development and resources available. If we had more time, I would:

  • Finish high-fidelity prototypes for all the main pages.
  • Conduct user testing
